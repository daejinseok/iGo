# 간단한 사용법

아래와 같은 내용으로 index.txt파일을 만든다.
subl|C:\bin\app\Sublime3\sublime_text.exe
...
( typeandrun config.ini 파일을 그대로 가져오면 된다. 단 파라메터는 아직 처리안해줌 ㅎㅎ)

단축키는 : alt + ; ( 아직은 설정 불가 )
ESC : 안보이도록
/Quit : 종료

버그가 너무 많아서 적기도 힘듬 ㅎㅎ
